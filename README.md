# atomic-option

> An atomic, nullable, owned pointer.

## [Documentation](https://crates.fyi/crates/atomic-option/0.1.0)

## Usage

Use the crates.io repository; add this to your `Cargo.toml` along
with the rest of your dependencies:

```toml
[dependencies]
atomic-option = "0.1"
```
